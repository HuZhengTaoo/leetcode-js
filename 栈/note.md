## 栈

- 栈是一种遵从后进先出（LIFO）原则的有序集合。新添加的或待删除的元素都保存在栈的
同一端，称作栈顶，另一端就叫栈底。在栈里，新元素都靠近栈顶，旧元素都接近栈底。

```js

function Stack(){
  let items =[]

  this.pop = function(){}
  this.peek = function(){}
  this.isEmpty = function(){}
  this.clear = function(){}
  this.size = function(){}
}
```
